<h3>Descripción General</h3>
<p>El programa es una implementación de un Red-Black Tree en Prolog, utilizando 100 nombres de estudiantes con sus respectivos numeros de carnet como ejemplo. SWI-Prolog brinda una libreria "rbtrees" con metodos de inserción, eliminación y busqueda.</p>
